# Task Families

## Primary Families

- `basic_galaxy_operations`: histories, uploads, metadata, collections, and output retrieval
- `single_tool_execution`: one tool, clear inputs, small parameter surface
- `workflow_retrieval_and_execution`: discover, import, bind, run, and verify workflows
- `tutorial_grounded_execution`: use GTN to choose and adapt the right analysis path
- `optimization_and_parameter_search`: improve a metric or satisfy a threshold under budget
- `failure_recovery`: recover from mismatched inputs, wrong assumptions, or bad initial choices
- `provenance_and_reproducibility`: reconstruct what happened and justify it with artifacts

## Current Legacy Coverage

- legacy tasks currently seed the workflow, tutorial, optimization, and reproducibility families
- `basic_galaxy_operations` and dedicated `failure_recovery` tasks still need to be authored as first-class v1 tasks
